# MySupplyCo Next.js Starter Documentation

Welcome to the **Enhanced Medusa Next.js Starter** documentation! This project combines Medusa's powerful e-commerce backend with Next.js 15 and an advanced companion panel system.

## 🚀 Quick Start

Get started with our enhanced Medusa storefront:

- **[Getting Started](/getting-started)** - Set up your development environment
- **[Configuration](/configuration)** - Configure your storefront settings  
- **[Companion Panel](/companion-panel)** - Learn about our AI-powered panel system
- **[Integration Guide](/integration)** - Quick setup and integration

## ✨ Key Features

### 🛍️ Complete E-commerce Solution
- Full Medusa V2 integration
- Shopping cart with persistent state
- Stripe payment processing
- User accounts and order management

### 🤖 AI-Powered Companion Panel
- Smart shopping assistant
- Contextual help system
- Configurable panel behaviors
- Mobile and desktop optimized

### ⚡ Performance & Developer Experience
- Next.js 15 with App Router
- Server Components and streaming
- TypeScript throughout
- Tailwind CSS styling
